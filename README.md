## Setup & Running

### Prerequisites
- **Python 3.14+**: Optimized for the latest Python runtime.
- **PowerShell**: Recommended for Windows deployment.

### Installation
1. **Clone the Repository**:
   ```powershell
   git clone https://github.com/Dirigentrix/agenthack-wankel-os.git
   cd agenthack-wankel-os
   ```
2. **Environment Configuration**:
   - Copy `.env.example` to `.env`.
   - Configure `DASHSCOPE_API_KEY` and `UIPATH_ORCHESTRATOR_URL`.
3. **Dependency Installation**:
   ```powershell
   python -m pip install -r requirements.txt
   ```
4. **Execution**:
   ```powershell
   python main.py
   ```
